# 参考的github 项目

## 测试代理ip是否可用

*   https://github.com/stamparm/fetch-some-proxies
    *   思路：先创建一个有代理 ip 的 txt 文件，然后循环该文件，判断里面的ip 哪些可用
    *   注意：获取的 proxy 可能有重复的，不同的线程可能在同一时间拿到该 proxy，都去查询数据库，但是无记录，所以都去判断该 proxy 是否可用，如果可用，则同时存入数据库，导致数据重复。
